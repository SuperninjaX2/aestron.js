WHAT IS AESTRONJS

aestronjs is a framework aiming to be a beast in terms of simplifying of web development. it wants to simplify and make web development complex at the same time. it has a pre-built CSS framework that is both sleek, minimal, and responsive. aestronjs js simplifies JavaScript as it has pre built functions and its own JavaScript extension system. the xjsx. XJSX makes writing the logic in html easier and better with close to English like syntax that looks like tour normal every bad HTML code

How to get started

set up

to install download the aestronjs library from the link.
Then unzip the package.

after unzipping make a folder named dist in your and copy the packages to that folder.

open your editor, and open your html file. link the aestronjs.css stylesheet to your HTML . Example below. 

        
     link rel="stylesheet" href="dist/aestron.css"

then link the JavaScript file in the head tag like in the example below. script src="dist/aestron.js"
