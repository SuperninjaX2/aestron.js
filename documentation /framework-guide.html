AestronJs Framework

The DomManipulator framework provides methods to manipulate DOM elements using vanilla JavaScript. In this guide, we will go through the methods for adding and removing classes, toggling classes, setting text and HTML content.

Adding and Removing Classes

The `addClass(className)` method adds the specified class to all elements matched by the selector passed in the constructor. The `removeClass(className)` method removes the specified class from all matched elements.

To use these methods, create an instance of the `DomManipulator` class with a CSS selector:

const elements = $('.my-class');

To add a class, call the `addClass` method:

elements.addClass('new-class');

To remove a class, call the `removeClass` method:

elements.removeClass('old-class');

Toggling Classes

The `toggleClass(className)` method adds the specified class to all elements matched by the selector passed in the constructor if the class is not present, and removes it if it is present.

elements.toggleClass('active');

Setting Text and HTML Content

The `text(value)` method sets the text content of all matched elements to the specified value. If no value is passed, the text content of the first matched element is returned.

elements.text('Hello, world!');

The `html(value)` method sets the HTML content of all matched elements to the specified value. If no value is passed, the HTML content of the first matched element is returned.

elements.html('<p>This is a paragraph.</p>');

Introduction to on() and off() methods

In JavaScript, the on() and off() methods are used to attach and detach event listeners to elements, respectively. Here's a simple example of how to use these methods:

To attach a click event listener to a button:

 
const myButton = document.querySelector('button');

myButton.on('click', function() {
  console.log('The button was clicked!');
});
   <

To remove the click event listener from the button:

 
myButton.off('click', function() {
  console.log('The button was clicked!');
});

In both cases, the first argument to the method is the name of the event you want to listen for (in this case, "click"). The second argument is a callback function that will be executed when the event is triggered.

Introduction to css(), appendElement(), and getCssValue() methods

In JavaScript, the css(), appendElement(), and getCssValue() methods can be used to manipulate the style and content of HTML elements. Here are some simple examples of how to use these methods:

To change the background color of a button:


const myButton = document.querySelector('button');

myButton.css('background-color', 'red');

To append a new paragraph element to a div:


const myDiv = document.querySelector('div');

myDiv.appendElement('p', {}, 'This is a new paragraph.');

To get the font size of a heading:


const myHeading = document.querySelector('h1');
const fontSize = myHeading.getCssValue('font-size');
console.log(`The font size of the heading is: ${fontSize}`);

In each example, the first argument to the method specifies the property or attribute you want to manipulate or retrieve. The second argument (if any) specifies the value you want to set for the property or attribute. The appendElement() method takes two additional optional arguments: an object that specifies attributes for the new element, and a string that specifies the text content of the new element.

Explanation of mergeElement() method

In JavaScript, the mergeElement() method can be used to create a new HTML element and add it to the page. This method has many features that can be used to customize the element.

Here is an example of how to use the mergeElement() method to create a new div element with all the available features:

      
const myElement = $(document.querySelector('.my-element'));
myElement.mergeElement('div', {
  attributes: {
    'id': 'new-element',
    'class': 'new-class',
    'data-attribute': 'some-value'
  },
  text: 'Hello, world!',
  css: 'background-color: red; color: white;',
  textNode: '',
  classes: ['class1', 'class2'],
  dataset: {
    'data-attribute1': 'value1',
    'data-attribute2': 'value2'
  },
  events: {
    'click': () => alert('You clicked me!')
  },
  children: [],
  style: {
    'font-size': '18px',
    'font-weight': 'bold'
  },
  callback: (element) => console.log('New element created!', element),
  replace: false,
  targetSelector: null,
  code: ''
});

In this example, the mergeElement() method creates a new div element with the following features:

  • attributes: The new element has an ID of "new-element", a class of "new-class", and a custom data attribute with a value of "some-value".
  • text: The new element has text content of "Hello, world!".
  • css: The new element has a background color of red and text color of white.
  • textNode: Not used in this example.
  • classes: The new element has two additional classes, "class1" and "class2".
  • dataset: The new element has two custom data attributes with values of "value1" and "value2".
  • events: The new element has a click event listener that displays an alert message.
  • children: Not used in this example.
  • style: The new element has a font size of 18px and font weight of bold.
  • callback: A callback function is called after the new element is created, which logs a message to the console.
  • replace: Not used in this example.
  • targetSelector: Not used in this example.
  • code: Not used in this example.

DomManipulator Methods Guide

getPosition()

The getPosition() method returns the position of the first element in the DomManipulator object relative to its offset parent. Here's an example:

const elementPosition = element.getPosition();

The getPosition() method is useful when you need to know the position of an element in order to position another element relative to it, for example.

find()

The find() method returns a new DomManipulator object that contains all descendants of the elements in the original object that match the given CSS selector. Here's an example:

const descendants = parent.find('p');

The above code will create a new DomManipulator object that contains all <p> elements that are descendants of the parent element.
